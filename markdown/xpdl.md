> #### 功能介绍

- ###### 基础功能

	```java
	1. 【对战】指定任意场景精灵；
	2. 【星魂】自动刷满一背包内的精灵；
	3. 【对战】模式下下支持支持自动【捕捉】异种、高资、稀有精灵；
	4. 【对战】中有精灵满级后支持自动从仓库【关注】列表中更换低等级精灵；
	```

- ###### 高级功能

	```java
	1. 【自动垂钓】(支持自定义参数)；
	2. 【化石战魂】刷高资或异种，可以指定异种、高资、或稀有；
	3. 【永恒梦境】自动扫荡（"水帘洞"刷蟠桃大圣进化石，"观景高阁"刷死神小幽进化石，"表盘广场"刷经验果和星魂药水），扫荡过程支持自动捕捉异种、高资、或稀有精灵；
	4. 【永恒梦境】自动捕捉自持梦境专属精灵, 包括"化石鱼龙"、"蟠桃小小"、"箱庭卫士"、"炽灯蛸(含异种)"、"电波猫(含异种)"、"负匣灵(含异种)"、"钢针鸟(含异种)"、"浮游卫士(含异种)"、"狂野波尼(含异种)"、"骑士小幽(含异种)"、"提灯沙尔(含异种)"、"涂鸦小兵(含异种)"；
	5. 更多功能，敬请期待······
	```
	

> #### 注意事项

- 【垂钓配置】：

	- 【垂钓功能】需要移动到【垂钓位点】附近然后再启动脚本；
	- 【预设参数】中的【按下时间】与【间隔时间】（均以毫秒为单位）小于0时将采用脚本内部的默认设置；
- 【精灵配置】：

  - 【活动区域】是指精灵在当前界面的活动范围，限制的越小识别的越快越准；
  - 【特征颜色】可通过【色值】后面的【瞄准器】从屏幕上选取颜色，【双击精灵】可以【放大取色】，取色时尽量选取目标精灵独有的特征色，如果精灵的特征颜色环境相似，可以将【相似度】设置为100（默认99）；
- 【对战配置】中的【异种捕捉】、【高资捕捉】、【稀有捕捉】影响多个功能：
	1. 【对战】模式下影响是否捕捉遇到的异种、高资、或稀有精灵；
	2. 【星魂】模式下将自动禁用【自动捕捉】功能（【异种暂定】不受影响）；
	3. 【化石战魂】模式下影响要刷的是异种、高资、还是稀有类型，如果只想抓异种化石战魂，则需要勾选【异种捕捉】且取消勾选【高资捕捉】以及【稀有捕捉】；
	4. 【永恒梦境】模式下影响是否捕捉梦境中的异种、高资、或稀有精灵，如果只想抓异种精灵，则需要勾选【异种捕捉】且取消勾选【高资捕捉】、【稀有捕捉】以及【盲目捕捉】；
